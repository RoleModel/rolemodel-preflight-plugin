# RoleModel Preflight

RoleModel Preflight is a standalone Framer plugin for running project checks before publishing. It scans canvas components, code files, links, copy, contrast candidates, and spacing templates from inside the Framer UI.

## Local Development

```sh
pnpm install
pnpm run dev
```

Open the development plugin in Framer from `https://localhost:5173/framer.json`.

## Checks

```sh
pnpm run typecheck
pnpm run test
pnpm run check
pnpm run build
```

The plugin ID is `rmpflt`.
